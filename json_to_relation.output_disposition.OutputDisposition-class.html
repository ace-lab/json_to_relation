Known Subclasses:

Specifications for where completed relation rows should be deposited, and in which format. Source options are files, stdout, and a MySQL table. This class is abstract, but make sure the subclasses invoke this super's __init__() when they are initialized.

Also defined here are available output formats, of which there are two: CSV, and SQL insert statements.
