Once at that page, download the three .csv files there. Your username is 'instructor', your password is 'myClass' (case sensitive). To download a file, right-click on the file name, and choose Save Link As..., or similar instruction (browser-dependent).

The main table file will include 'EventXtract' in its name. Each of its rows contains one 'event,' a student's interaction with the OpenEdX learning platform. The table file that includes 'VideoInteraction' excerpts rows from the EventXtract table that are video related. Similarly, rows in the table file that contains the name 'ActivityGrade' excerpts all the assignment related rows from EventXtract. These two tables are conveniences.

For an explanation of the table columns, please visit our how-to page.
